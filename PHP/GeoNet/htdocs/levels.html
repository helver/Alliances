Access Levels
Customers Have access to Color Block, Map, Alarm History, and Tid History pages.
Can not see element IP addresses.
Internal Customers Can see element IP addresses.
Users Have access to Database Update pages.
Cannot create or modify UserGroups.
Can acknolwedge alarms.
Admin Can create or modify UserGroups.
Can assign customers to UserGroups.
Can add or modify Users.
Cannot set a user's access level to Admin or SuperAdmin.
Cannot change the access level of any Admin or SuperAdmin.
Super Admin Can set a user's access level to Admin or SuperAdmin.
 
Each access level retains the abilities of the access levels below it, unless otherwise specified.
