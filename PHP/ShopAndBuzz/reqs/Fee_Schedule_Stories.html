Fee Schedule:

Fee Schedule is the algorithm that calculates the amount and timing of the monthly Buzz Fee that is taken from Seller's Paypal accounts and transferred to the Buzz Paypal account.

Admin can set Fee Schedule start date for any Seller.

Admin can set Fee Schedule monthly fee for any Seller.

Admin can set a trial period duration and monthly fee for any Seller.

Inactive Sellers will not be subject to Fee Schedule.
