Role

Registered BuzzUsers will be acting as a particular Role when using the Buzz System. Permissions and capabilities will be granted to roles. Users will be assigned one or more roles. When performing Buyer related activities, the user will be acting with the Buyer role (if they have it). When performing Seller related activities, the user will be acting with the Seller role (if they have it). When performing Admin related activities, the user will be acting with the Admin role (if they have it). Otherwise, the user will be acting with the BuzzUser role.
Roles:
  • Visitor
  • Buyer
  • Seller
  • Admin
  • BuzzUser
